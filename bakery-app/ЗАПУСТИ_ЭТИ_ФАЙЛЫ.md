# Запусти эти файлы по порядку

## 1. Установи react-native-reanimated

Запусти файл:
```
2-install-reanimated.bat
```

Или вручную:
```bash
npm install react-native-reanimated
```

## 2. Включи длинные пути Windows (ВАЖНО!)

Открой PowerShell **как администратор** и выполни:

```powershell
New-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\FileSystem" -Name "LongPathsEnabled" -Value 1 -PropertyType DWORD -Force
```

После этого **перезагрузи компьютер**.

## 3. Собери APK

Запусти файл:
```
3-clean-and-build.bat
```

Или вручную:
```bash
cd android
gradlew clean
cd ..
npx expo prebuild --clean
cd android
gradlew assembleRelease
```

## Готовый APK будет здесь:

```
android\app\build\outputs\apk\release\app-release.apk
```

## Если всё равно ошибка с длинными путями:

Перемести весь проект в корень диска:
```
C:\bakery\
```

Чтобы пути были короче.
