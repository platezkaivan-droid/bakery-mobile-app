# ✅ Финальные доработки

## Что исправлено

### 1. Модалка подтверждения отмены теперь поверх деталей заказа
**Проблема:** Модалка "Отменить заказ?" появлялась под карточкой деталей заказа

**Решение:** Перенесли модалку ВНУТРЬ компонента деталей заказа

**До:**
```
<Screen>
  <OrderDetailsModal />  ← Сверху
  <CancelConfirmModal /> ← Снизу (не видно)
</Screen>
```

**После:**
```
<OrderDetailsModal>
  ... детали заказа ...
  <CancelConfirmModal /> ← Внутри, поверх всего
</OrderDetailsModal>
```

**Стили:**
- `position: 'absolute'` - абсолютное позиционирование
- `zIndex: 9999` - максимальный приоритет
- `elevation: 10` - для Android
- Полупрозрачный фон `rgba(0,0,0,0.7)`

### 2. Кнопка "Очистить всё" для удаления старых заказов
**Проблема:** Старые тестовые заказы остались в БД

**Решение:** Добавлена кнопка корзины в header

**Функционал:**
- Удаляет ВСЕ заказы текущего пользователя из БД
- Показывает уведомление "Все заказы удалены"
- Кнопка появляется только если есть заказы
- Красная иконка корзины на светлом фоне

## Как использовать

### Удалить все заказы
1. Откройте вкладку "Заказы"
2. Нажмите на иконку корзины в правом верхнем углу
3. Все заказы будут удалены из БД
4. Список обновится автоматически

### Отменить конкретный заказ
1. Откройте детали заказа
2. Нажмите "Отменить заказ" (если заказ активный)
3. Появится модалка подтверждения ПОВЕРХ деталей
4. Выберите "Да, отменить" или "Нет, оставить"
