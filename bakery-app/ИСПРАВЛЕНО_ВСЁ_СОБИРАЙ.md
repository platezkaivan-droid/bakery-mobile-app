## После перезагрузки:

### Вариант 1: Сборка в Android Studio (РЕКОМЕНДУЕТСЯ)

1. Открой проект в Android Studio:
   ```
   C:\Users\Administrator\Documents\bakery-app\bakery-mobile-app\bakery-app\android
   ```

2. Build → Generate Signed Bundle / APK → APK

3. Выбери Release

4. Создай keystore (если ещё не создал):
   - Заполни хотя бы одно поле Certificate (например First and Last Name)
   - Запомни пароли!

5. Жди сборку

6. APK будет в: `android/app/build/outputs/apk/release/app-release.apk`

### Вариант 2: Через командную строку

```bash
cd bakery-app/bakery-mobile-app/bakery-app/android
gradlew assembleRelease
```

## Если всё равно ошибка с google-services.json:

Проверь что в Firebase Console добавлен package name: `com.bakery.pavlova`

Или временно закомментируй в `android/app/build.gradle`:
```gradle
// apply plugin: 'com.google.gms.google-services'
```

## Проверка после сборки:

APK должен быть здесь:
```
android/app/build/outputs/apk/release/app-release.apk
```

Размер: ~50-80 MB
