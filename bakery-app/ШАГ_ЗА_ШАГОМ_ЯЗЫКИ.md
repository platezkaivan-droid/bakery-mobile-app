# 📖 ШАГ ЗА ШАГОМ: Исправление татарского и армянского языков

## Проблема
При выборе татарского или армянского языка товары остаются на русском.

---

## 🎯 Решение

### ШАГ 1: Откройте Supabase

1. Перейдите на сайт: **https://supabase.com**
2. Войдите в свой аккаунт
3. Выберите ваш проект (bakery-app)

```
┌─────────────────────────────────────┐
│  🌐 Supabase Dashboard              │
│                                     │
│  📁 Projects                        │
│    └─ 🍰 bakery-app  ← Выберите   │
└─────────────────────────────────────┘
```

---

### ШАГ 2: Откройте SQL Editor

В левом меню найдите и нажмите **SQL Editor**:

```
┌─────────────────────┐
│ 📊 Dashboard        │
│ 🗄️  Table Editor    │
│ 📝 SQL Editor  ← ТУТ│
│ 🔐 Authentication   │
│ 📦 Storage          │
└─────────────────────┘
```

---

### ШАГ 3: Откройте файл с переводами

На вашем компьютере найдите файл:
```
bakery-app/
  bakery-mobile-app/
    bakery-app/
      ЗАПОЛНЕНИЕ_ПЕРЕВОДОВ_ВСЕХ_ТОВАРОВ.sql  ← Этот файл
```

Откройте его в любом текстовом редакторе (Блокнот, VS Code, и т.д.)

---

### ШАГ 4: Скопируйте весь SQL-код

1. Нажмите **Ctrl+A** (выделить всё)
2. Нажмите **Ctrl+C** (скопировать)

```sql
-- Вы должны скопировать весь файл, начиная с:
-- ============================================
-- ПОЛНОЕ ЗАПОЛНЕНИЕ ПЕРЕВОДОВ ВСЕХ ТОВАРОВ
-- ============================================

UPDATE products SET 
  name_en = 'Chocolate Croissant',
  name_kz = 'Шоколадты круассан',
  name_tt = 'Шоколадлы круассан',
  name_uz = 'Shokoladli kruassan',
  name_hy = 'Շոկոլադե կրուասան',
  ...
```

---

### ШАГ 5: Вставьте в SQL Editor

1. Вернитесь в Supabase SQL Editor
2. Нажмите **Ctrl+V** (вставить)
3. Вы должны увидеть весь SQL-код в редакторе

```
┌─────────────────────────────────────────┐
│ SQL Editor                              │
├─────────────────────────────────────────┤
│ -- ПОЛНОЕ ЗАПОЛНЕНИЕ ПЕРЕВОДОВ         │
│ UPDATE products SET                     │
│   name_en = 'Chocolate Croissant',     │
│   name_tt = 'Шоколадлы круассан',      │
│   ...                                   │
│                                         │
│ [Здесь будет весь ваш SQL-код]         │
└─────────────────────────────────────────┘
```

---

### ШАГ 6: Выполните SQL-скрипт

Нажмите кнопку **Run** (или **Ctrl+Enter**):

```
┌─────────────────────────────────────────┐
│ [▶ Run]  [Save]  [Format]              │
└─────────────────────────────────────────┘
```

Подождите 5-10 секунд...

---

### ШАГ 7: Проверьте результат

Вы должны увидеть сообщение об успехе:

```
✅ Success. No rows returned
```

Или:

```
✅ 24 rows updated
```

Это значит что переводы успешно добавлены!

---

### ШАГ 8: Перезапустите приложение

1. **Полностью закройте** приложение (не просто свернуть)
2. Откройте приложение заново
3. Перейдите в **Профиль → Язык**
4. Выберите **Татарский** или **Армянский**
5. Вернитесь на главную страницу

---

## ✅ Готово!

Теперь все товары должны отображаться на выбранном языке!

### Что будет переведено:
- ✅ Названия товаров
- ✅ Описания товаров
- ✅ Товары в корзине
- ✅ Товары в избранном
- ✅ Товары в заказах

### Примеры:

**Круассан с шоколадом:**
- 🇷🇺 Русский: "Круассан с шоколадом"
- 🇷🇺 Татарча: "Шоколадлы круассан"
- 🇦🇲 Հայերեն: "Շոկոլադե կրուասան"

**Хлеб ржаной:**
- 🇷🇺 Русский: "Хлеб ржаной"
- 🇷🇺 Татарча: "Арыш икмәге"
- 🇦🇲 Հայերեն: "Գարեջրային հաց"

**Синнабон:**
- 🇷🇺 Русский: "Синнабон классический"
- 🇷🇺 Татарча: "Классик дарчин булкасы"
- 🇦🇲 Հայերեն: "Դասական դարչնով բուլկա"

## ❓ Если что-то пошло не так

### Ошибка: "column does not exist"
Сначала выполните этот SQL:
```sql
ALTER TABLE products ADD COLUMN IF NOT EXISTS name_tt TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS description_tt TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS name_hy TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS description_hy TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS name_en TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS description_en TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS name_kz TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS description_kz TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS name_uz TEXT;
ALTER TABLE products ADD COLUMN IF NOT EXISTS description_uz TEXT;
```

Затем повторите ШАГ 6.

### Товары всё ещё на русском
1. Убедитесь что SQL выполнился успешно (зелёная галочка)
2. Полностью закройте и откройте приложение
3. Проверьте что выбран правильный язык в настройках

### Нужна помощь?
Напишите в поддержку с скриншотом ошибки из SQL Editor.
