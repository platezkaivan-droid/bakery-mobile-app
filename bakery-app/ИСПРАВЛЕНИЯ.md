# История исправлений

## Проблема 1: Ошибка "Unable to resolve react-native-web/dist/index"

**Причина**: Отсутствовали пакеты для веб-версии или были установлены неправильные версии.

**Решение**:
```bash
# Удалить старые пакеты
npm uninstall react-native-web react-dom

# Установить через expo install (автоматически подберет правильные версии)
npx expo install react-native-web react-dom
```

## Проблема 2: Ошибка "(0, _expoModulesCore.registerWebModule) is not a function"

**Причина**: Несовместимость версий `expo-font` и `expo-modules-core`.

**Решение**:
```bash
# Переустановить проблемные пакеты
npx expo install expo-font expo-modules-core

# Запустить с очисткой кеша
npx expo start -c
```

## Рекомендации

1. Всегда используйте `npx expo install` вместо `npm install` для установки пакетов Expo
2. При возникновении ошибок совместимости, запускайте с флагом `-c` для очистки кеша
3. Если проблемы сохраняются, удалите `node_modules` и `package-lock.json`, затем выполните `npm install`
